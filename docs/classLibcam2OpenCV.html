◆ OnFrame

using Libcam2OpenCV::OnFrame = std::function<void(const cv::Mat &, const libcamera::ControlList &)>

The callback prototype carrying the current openCV frame. This callback needs to be defined by you and usually as a lambda function: [&](const cv::Mat &mat, const libcamera::ControlList &) { myImageProcessor(mat); }); which is then registered with registerCallback.
